# 🍽️ Meet & Eat – Food Event Web App

## 📌 Overview
Meet & Eat is a Flask-based web application that allows users to create, browse, and manage food-related events. It connects people who want to share meals and participate in food meetups.

---

## 🚀 Features
- Create food events
- View list of events
- Simple and clean UI
- Flask backend integration
- MongoDB database support

---

## 🛠️ Tech Stack
- Python
- Flask
- HTML, CSS
- MongoDB
- PyMongo

---

app.py

    index.html
    create_event.html
    events.html

## ⚙️ How to Run Locally

1. Clone the repository:
git clone https://github.com/riyasahu7/meet-and-eat.git

2. Go to project folder:
cd meet-and-eat

3. Create virtual environment:
python -m venv venv

4. Activate it:
venv\Scripts\activate

5. Install dependencies:
pip install flask flask-pymongo

6. Run the app:
python app.py
